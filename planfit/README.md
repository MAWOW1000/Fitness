### commit 컨벤션

```
- **Feature** : 기능 추가
- **Fix** : 에러 수정, 버그 수정
- **Refactor** : 코드 리펙토링 (기능 변경 없이 코드만 수정할 때)
- **Chore** : 코드 수정 (기능의 변화가 있을 때)
- **Setting** : gradle 세팅, 위의 것 이외에 거의 모든 것
- **Merge** : main 머지할 때
```
